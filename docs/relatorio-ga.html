1. Descrição do Tema Escolhido

O tema escolhido foi o Blog de Tecnologia e Programação (Tema 5). O DevBlog é um blog técnico voltado para estudantes e desenvolvedores, com artigos sobre desenvolvimento web, mobile, inteligência artificial, segurança, DevOps e ciência de dados.

O blog foi pensado como um espaço informativo e educacional, com conteúdo original sobre temas relevantes à área de formação dos integrantes. O design utiliza um tema escuro inspirado no GitHub, oferecendo uma experiência visual agradável e familiar para desenvolvedores.

6. Tecnologias e Técnicas Utilizadas

HTML

  • HTML5 semântico com tags <header>, <nav>, <main>, <section>, <article> e <footer>
  • Formulários com <input>, <textarea>, <select> e validação HTML nativa (required)
  • Hierarquia de cabeçalhos <h1> a <h3>
  • Imagens com atributo alt descritivo para acessibilidade
  • Links internos (entre páginas) e externos (com target="_blank" e rel="noopener noreferrer")
  • Âncoras internas na página de categorias para navegação suave
  • Uso de <time datetime> para datas semânticas nos artigos

CSS

  • Arquivo CSS externo único (css/style.css)
  • CSS Grid para layouts de artigos, categorias, ferramentas e equipe
  • Flexbox para header, navegação e componentes menores
  • Tema escuro com paleta consistente (inspirada no GitHub Dark)
  • Media queries para responsividade em 768px e 480px
  • Transições CSS para hover em cards e links
  • Menu hamburger responsivo para mobile (toggle via onclick)
  • Header sticky com position: sticky

7. Dificuldades Encontradas e Soluções

Dificuldade Solução
Manter a consistência visual em 11 páginas diferentes Uso de um único arquivo CSS externo com classes reutilizáveis e paleta de cores definida em todo o site
Layout responsivo para diferentes tamanhos de tela Uso de CSS Grid com auto-fill e minmax() para grids adaptáveis, complementado por media queries em 768px e 480px
Menu de navegação funcional em mobile sem JavaScript complexo Implementação de menu toggle com evento onclick inline que alterna classe CSS para mostrar/esconder a navegação
Garantir que todos os links levassem a conteúdo real Criação de 6 artigos completos individuais e categorias com âncoras internas, eliminando qualquer link vazio
Formatação rica nos artigos (código, blockquotes, listas) Estilização CSS dedicada para os elementos dentro de .article-body com cores de sintaxe e espaçamento adequado

9. Declaração de Uso de IA

Ferramenta utilizada: GitHub Copilot (assistente de código baseado em IA)

O que foi gerado: A estrutura inicial das páginas HTML e do arquivo CSS foi gerada com auxílio do GitHub Copilot, incluindo a estrutura semântica, estilização base e layout responsivo.

Como foi adaptado: Todo o código gerado foi revisado, compreendido e adaptado pelo grupo. Os conteúdos dos artigos foram revisados para garantir precisão técnica. A paleta de cores, escolha das categorias e organização das seções foram definidas pelo grupo. O grupo é capaz de explicar cada parte do código entregue.
